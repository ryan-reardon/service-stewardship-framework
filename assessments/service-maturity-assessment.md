# Service Maturity Assessment Questionnaire
## Enterprise Technology Service Stewardship Framework

This consolidated assessment evaluates maturity across all 12 domains.

Scoring Scale:
1 = Reactive / Informal  
2 = Managed / Documented but inconsistent  
3 = Optimized / Measured and governed  
4 = Predictive / Proactive and continuously improving  

Evidence should be cited for each score.

---

## 1. Strategic Stewardship
- Documented multi-quarter roadmap aligned to enterprise strategy
- Initiatives tied to measurable business outcomes
- Lifecycle stage explicitly defined
- Modernization capacity protected
- Cost-to-value trend reviewed quarterly

## 2. Organizational Capability
- Documented team charter and ownership boundaries
- Skills matrix aligned to roadmap and ops needs
- Redundancy for critical responsibilities
- Sustainable on-call workload
- Active growth and succession planning

## 3. Value Execution
- Explicit value-based prioritization
- Predictable delivery cadence
- NFRs included in definition of done
- Dependencies actively managed
- Post-delivery outcome measurement

## 4. Engineering Excellence
- CI/CD with enforced quality gates
- Infrastructure as code
- Change failure rate tracked
- Toil measured and reduced
- Secure SDLC embedded

## 5. Operational Reliability
- SLAs/SLOs defined and measured
- Error budgets influence change decisions
- Postmortem actions tracked
- Alert noise actively reduced
- Capacity planning proactive

## 6. Architectural Integrity
- Architecture documented and current
- ADRs used
- Architectural debt tracked
- NFRs evaluated in reviews
- Tooling sprawl governed

## 7. Service Experience
- Support model documented
- Response/resolution targets defined
- Repeat issues reduced
- Knowledge base maintained
- Customer feedback influences roadmap

## 8. Security & Risk Governance
- Complete dependency inventory
- Vulnerability SLAs enforced
- Patch cadence consistent
- Access reviews conducted
- Compliance evidence automated

## 9. Financial Stewardship
- Full TCO visibility
- Unit cost tracked
- Forecast updated quarterly
- Cost optimization backlog maintained
- Vendor/license governance active

## 10. Observability & Intelligence
- SLOs instrumented and reliable
- Alert quality high
- Tracing for critical flows
- Telemetry cost governed
- Forecasting based on telemetry trends

## 11. Innovation & Modernization
- Technical debt categorized and scored
- Modernization capacity protected
- Legacy components tracked
- Experiments structured and measured
- Modernization outcomes quantified

## 12. Portfolio Alignment
- Dependency map current
- Integration contracts versioned
- Architecture compliance governed
- Duplication reviewed
- Cross-service risks tracked
